Boiling City Via Instagram

This is prof of concept in how to consume realtime data from Instagram.

Demo

Thanks to heroku.com I managed to deploy it at http://boiling-city-instagram.herokuapp.com

How it works?

The application is reading the configuration variables from /server/config.js Based on _cityCoverage, tagSubscription _ it creates subscriptions for media. When a new photo is uploaded the instagram posts to the application the media information and the app will push the media to browsers that are connected via sockets.

How to configure?

  • Open file /server/config.js
  • Add your domain to the variable domainName
  • Add your Instagram clientId and clientSecret to the instagramConfig object. (reference: Instagram Developer API

How to run it?

node app.js

For developers:

  • It creates subscription automatically when the first socket.io receives a new connection.
  • It unsubscribes automatically when there is no one connected via socket.io
  • It creates a cache from the first images that were search based on the tagSubsciption array and it will update that array every-time a new media was pushed to the browser.
